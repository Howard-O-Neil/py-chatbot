# Quickstart

```bash
  # development only

  make bootstrap

  source env/bin/activate.fish  # if u use fish shell
  source env/bin/activate       # if u use bash shell
  # search google               # is u use other shell

  make set-dns
  make auth-script
  docker-compose up
```
